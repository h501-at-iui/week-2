# Week 2

The goal of this lab is to give you the tools you need to start building robust functions which can make useful calculations on large datasets.

To that end, we'll cover the following:

- More Python Basics
- Loops
- Generator Iterators
- NumPy
- NumPy Arrays
- Array Operations
